## Description
This is the React-based frontend for an AI-powered blog platform. It connects to the Express/MongoDB backend to fetch, display, create, edit, and delete blog posts. Features include image uploads (via ImageKit API) and AI-generated summaries using Google Gemini.




## Features
- Display blog posts with AI-generated summaries
- Create, edit & delete posts
- Upload images via ImageKit
- Responsive design for mobile & desktop
- (Optional) Authentication and user profiles



## Tech Stack
- React 18
- Axios for HTTP requests
- Tailwind CSS (or your preferred styling)
- React Router
- (Optional) Redux or Context API
